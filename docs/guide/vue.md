## 安装

:::code-group

```bash [npm]
$ npm i @vrx-arco/icons-vue
```

```bash [yarn]
$ yarn add @vrx-arco/icons-vue
```

```bash [pnpm]
$ pnpm add @vrx-arco/icons-vue
```

```bash [bun]
$ bun add @vrx-arco/icons-vue
```
